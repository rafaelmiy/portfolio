FPL BR - EFB
(Electronic Flight Bag)

Desafio

Agregar informações importantes para pilotos aeronáuticos, o aplicativo contempla previsão do tempo, documentos oficiais aeronáuticos (cartas aeronáuticas) atualizados, integração do plano de voo (documento que contém informações a respeito de um voo) com o mapa de planejamento e gerenciamento de planos de voo.

O Time

Contou com o apoio de profissionais multidiciplinares como UX e UI designers, product owner, analistas de requisitos, engenheiros, pilotos, entre outros.

Interface

O aplicativo foi prototipado para celulares e tablets (Android e iOS), também adaptado para ambientes com baixa luminosidade, como dentro de uma cabine de avião com o modo noturno (dark mode). Para validação das telas, foram feitos testes com usuários utilizando protótipos navegáveis de alta fidelidade.
